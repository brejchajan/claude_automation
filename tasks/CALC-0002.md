## Description

Implement a simple command line calculator in pure C, compilable with macOS
clang. The calculator runs as an interactive REPL (read-eval-print loop). On
startup it prints a welcome message and a prompt. The user types a mathematical
expression (numbers and operators such as +, -, *, /, parentheses), presses
Enter, and the result is printed to the screen. The calculator keeps accepting
new expressions until the user types "exit" or "quit" (case-insensitive), or
sends EOF (Ctrl+D).

The implementation must use a recursive descent parser to safely evaluate
expressions. Do NOT use `system()`, `popen()`, or any shell execution to
evaluate input. The parser must support operator precedence:
- Parentheses (highest)
- Unary minus
- `*`, `/` (left-to-right)
- `+`, `-` (left-to-right)

The project structure:
- `calculator.c` -- main source file with the REPL, tokenizer, and parser
- `Makefile` -- builds the project with `clang`, using `-Wall -Wextra -Werror
  -std=c11`. The default target produces a `calculator` binary. Include a
  `clean` target.
- `test_calculator.sh` -- a shell-based test script that feeds expressions to
  the calculator binary via stdin/pipe and asserts expected output. Must exit
  with code 0 on success, non-zero on failure.

## Acceptance Criteria

- Running `make` produces a `calculator` binary with zero warnings.
- Running `./calculator` starts an interactive prompt.
- Entering `2 + 3` prints `5`.
- Entering `10 / 3` prints `3.333333` (or similar float representation).
- Entering `2 * (3 + 4)` prints `14`.
- Entering `(1 + 2) * (3 + 4)` prints `21`.
- Entering `-5 + 3` prints `-2` (unary minus support).
- Entering invalid input like `hello` prints a user-friendly error message.
- Division by zero prints an error message instead of crashing.
- Typing `exit` or `quit` ends the program gracefully.
- Pressing Ctrl+D (EOF) ends the program gracefully.
- `make clean` removes the binary and any build artifacts.
- `bash test_calculator.sh` runs all tests and exits with code 0.
- The code compiles cleanly with `clang -Wall -Wextra -Werror -std=c11`.
- No use of `system()`, `popen()`, `eval`, or any shell-based evaluation.
